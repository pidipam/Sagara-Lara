## Deskripsi

*Sagara Lara* adalah game hyper-casual dengan mekanik utama typing yang menggambarkan perjalanan batin William, seorang anak yang tumbuh dengan kecemasan, tekanan, dan kehilangan tulang punggung emosionalnya: ibunya. Setelah kematian ibunya, pikirannya menjadi lautan gelap yang perlahan menelan damai kecil yang ia miliki.

Laut di game ini adalah representasi langsung dari kondisi mental William, lengkap dengan ombak keras, bisikan jahat, dan makhluk yang lahir dari rasa bersalah dan ketakutannya.

Mercusuar adalah cahaya terakhir yang menghubungkannya dengan ibunya, simbol harapan dan penyembuhan.

Perjalanan menuju penyembuhan penuh dengan rintangan, ketidakpastian, bagaikan terombang-ambing di samudra yang luas. Badai menerjang, angin bertiup kencang, ombak menghantam dengan keras, setiap detik terasa menjadi akhir dari segalanya. 

## Mekanisme
Pemain mengetik kata-kata negatif untuk meredam ombak pikirannya. Setiap wave melambangkan fase yang harus ia lewati.  Setiap boss adalah manifestasi trauma yang harus dia hadapi.

Dan ketika ia menang, suara ibunya kembali…

Tetapi lautan akan memanggilnya lagi…

Akankah William menemukan matahari terbitnya lagi? Apakah ia akan menemukan cahaya yang dapat menerangi hari-harinya lagi? Atau ia akan ditelan oleh dalamnya lautan dan diseret oleh derasnya ombak?

## Setup

### Prasyarat
* Unity Hub & Unity Editor 2022.3.62f1.
* Git LFS

### 
1.  **Clone Repository:**
    ```bash
    git clone [https://github.com/pidipam/Sagara-Lara.git](https://github.com/pidipam/Sagara-Lara.git)
    cd Sagara-Lara
    ```

2.  **Pull LFS Assets:**
    Jika aset gagal:
    ```bash
    git lfs pull
    ```

3.  **Unity:**
    * Buka Unity Hub.
    * Klik *Add* -> Pilih folder `Sagara-Lara`.
